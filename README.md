# 🐍 Snake Game

**Snake Game** is a classic arcade-style game built using **HTML, CSS, and JavaScript**. Players control a snake that grows longer as it collects food while avoiding collisions with walls and itself. The project recreates the timeless gaming experience with a simple and responsive interface.

## ✨ Features

* 🐍 Classic snake gameplay
* 🍎 Food collection and score tracking
* 📈 Snake grows as food is consumed
* 🎮 Keyboard controls
* ⚡ Smooth and responsive gameplay
* 📱 Lightweight and user-friendly design

## 🔄 How It Works

1. 🚀 Start the game.
2. 🎮 Control the snake using keyboard inputs.
3. 🍎 Collect food to increase your score.
4. 📈 Watch the snake grow longer with each item collected.
5. ⚠️ Avoid hitting walls or the snake's own body.
6. 🏆 Try to achieve the highest score possible.
